- **[DoodleCSS](https://github.com/chr15m/DoodleCSS)** — 手绘涂鸦风格主题
- **GitHub Pages** — 免费托管

## ✏️ 如何自定义

1. **替换占位内容** — 编辑各 HTML 文件中的示例文字和图片
2. **添加真实作品** — 在 `works.html` 中添加你的作品卡片
3. **嵌入视频** — 在 `videos.html` 中替换 B站/YouTube 嵌入代码
4. **更新个人信息** — 修改 `about.html` 中的头像、简介和社交链接
5. **添加图片** — 将图片放入 `assets/images/` 目录
